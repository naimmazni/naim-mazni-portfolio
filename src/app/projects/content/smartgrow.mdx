## Technical Architecture

### Hardware Layer

**ESP32 Microcontroller**

- WiFi connectivity for data transmission
- Multiple sensor inputs
- Low power consumption
- Reliable outdoor operation

**Sensor Array**

- Capacitive soil moisture sensor
- DHT22 temperature/humidity sensor
- BH1750 light sensor
- pH sensor module
- Water level sensor

### Mobile Application

Built with Flutter for cross-platform deployment:

- Beautiful, intuitive UI
- Real-time data updates
- Offline data caching
- Charts and data visualization
- Multi-language support

### Cloud Infrastructure

**AWS IoT Core**

- Secure device connectivity
- Real-time message routing
- Device shadow for state management
- Data storage in DynamoDB
- Lambda functions for processing

### AI Integration

**Gemini API** provides:

- Natural language insights
- Context-aware recommendations
- Image-based plant health analysis
- Predictive analytics for yield
- Personalized farming tips

## System Workflow

1. **Data Collection**: ESP32 sensors gather environmental data
2. **Transmission**: Data sent to AWS IoT Core via MQTT
3. **Processing**: Lambda functions process and store data
4. **Analysis**: Gemini API analyzes patterns and generates insights
5. **Visualization**: Flutter app displays data and recommendations
6. **Action**: Automated responses or farmer-triggered controls
