## Overview

The Azrina Mahdi Workshop Management System is a full-featured web application designed to streamline workshop operations from registration to payment processing. Built with Laravel 12 and modern frontend technologies, it provides both public-facing workshop listings and a comprehensive administrative dashboard for managing all aspects of workshops, registrations, and payments.

## Problem

Workshop organizers face challenges in managing multiple aspects of their business: tracking registrations, processing payments, scheduling workshop dates, monitoring capacity, and maintaining clear communication with participants. Manual management of these tasks is time-consuming, error-prone, and makes it difficult to scale operations or gain insights into business performance.

## Solution

This system provides an end-to-end solution that automates workshop management workflows. Participants can easily browse and register for workshops online, while administrators have access to a powerful dashboard with analytics, real-time notifications, and comprehensive management tools. The integrated Billplz payment gateway ensures secure and reliable payment processing with automatic confirmation.

## Key Features

### Public Features

- **Workshop Listing**: Browse available workshops with detailed information
- **Workshop Registration**: User-friendly registration form for participants
- **Payment Integration**: Secure payment processing with Billplz gateway
- **Email Notifications**: Automated confirmation emails upon registration
- **Workshop Calendar**: Visual calendar view of workshop dates and availability

### Admin Features

- **Dashboard Analytics**: Comprehensive overview with charts and key metrics
- **Workshop Management**: Create, edit, and manage workshops with images and pricing
- **Date Management**: Schedule and manage multiple dates per workshop
- **Registration Management**: View, filter, and export registration data
- **Payment Tracking**: Monitor and update payment statuses in real-time
- **Notification System**: Real-time notification center for admin activities
- **PDF Export**: Generate and download registration reports

## Technologies Used

- **Backend**: Laravel 12.x (PHP 8.2+) - Modern PHP framework for robust web applications
- **Frontend**: Tailwind CSS 3.x, Alpine.js 3.x - Utility-first CSS and reactive JavaScript
- **Database**: MySQL/MariaDB - Reliable relational database management
- **Authentication**: Laravel Breeze - Simple authentication scaffolding
- **PDF Generation**: DomPDF - Generate professional PDF reports
- **Charts**: Chart.js 4.x - Interactive data visualizations
- **Build Tool**: Vite 7.x - Fast frontend build tool
- **Email**: Laravel Mail with SMTP - Automated email notifications
- **Payment Gateway**: Billplz - Malaysian payment processing integration
- **Testing**: PestPHP - Modern PHP testing framework

## System Architecture

The application follows Laravel's MVC architecture:

- **Frontend Layer**: Blade templates with Tailwind CSS and Alpine.js for interactivity
- **Application Layer**: Laravel controllers handling business logic and routing
- **Database Layer**: Eloquent ORM for database interactions with MySQL
- **External Services**: Billplz API for payment processing, SMTP for emails
- **Asset Pipeline**: Vite for optimized asset compilation and hot module replacement
