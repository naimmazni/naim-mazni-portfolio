## Technical Implementation

### Frontend Architecture

The website uses Next.js 15 with the App Router architecture, providing excellent performance through server-side rendering and automatic code splitting. TypeScript ensures type safety throughout the codebase.

### Content Management

Projects are written in MDX format, allowing for rich content with embedded React components. This approach provides flexibility while maintaining clean separation of content and presentation.

### Styling System

SCSS modules combined with Once UI's design tokens create a maintainable and scalable styling architecture. The theme system supports light and dark modes with smooth transitions.

### Cal.com Integration

Custom-built CalendarButton component that:

- Initializes Cal.com embed API on page load
- Detects website theme changes in real-time
- Passes current theme to Cal.com modal dynamically
- Provides seamless booking experience
